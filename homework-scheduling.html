שאלה 3
פונקצית המטרה: max(wj·Tj)
1) האם מקובלת?
כן. Tj = max(Cj - dj, 0) → אם Cj גדל, Tj לא יורד. wj·Tj לא יורד ב-Cj. max של פונקציות לא יורדות → לא יורדת. → פונקציה מקובלת. ∎
2) איך למצוא סידור אופטימלי?
הבעיה הכללית NP-hard. אם wj = w לכל j: max(w·Tj) = w · max(Tj) = w · Tmax → מספיק למזער Tmax → סידור EDD. במקרה הכללי (wj שונים) אין אלגוריתם פולינומיאלי ידוע, צריך לבדוק סידורים או תכנות דינמי.
3) שינויים בנתונים
a. dj השתנו לערכים אחרים: הסידור עשוי להשתנות (תלוי ב-dj). ערך פונקצית המטרה ישתנה. b. כל dj קטנו ב-1: הסידור נשאר זהה — הסדר היחסי בין dj לא השתנה. ערך פונקצית המטרה יגדל — כי Lj = Cj-dj גדל ב-1 → Tj גדל (אם הג'וב כבר מפגר).
שאלה 4
1) הוכחה: tl≤tm, dl≤dm → l לפני m באופטימום
נניח בשלילה סידור S* אופטימלי בו m לפני l. S*: ... m, [B], l, ... (B = ג'ובים בין m ל-l) נבנה S' ע"י החלפת m ו-l: S': ... l, [B], m, ... נסמן A = סכום עיבוד של כל מה שלפני m. ב-S*: Cm = A + tm Cl = A + tm + ∑tB + tl ב-S': C'l = A + tl C'm = A + tl + ∑tB + tm (= Cl) שינויים בפיגור: ג'ובים ב-B: C'b = Cb - tm + tl ≤ Cb (כי tl≤tm) → T'b ≤ Tb ✓ ג'וב l: C'l = A+tl ≤ A+tm+∑tB+tl = Cl → T'l ≤ Tl ✓ ג'וב m: C'm = Cl, אז: T'm = max(Cl - dm, 0) Tl = max(Cl - dl, 0) dl ≤ dm → Cl-dl ≥ Cl-dm → Tl ≥ T'm סה"כ: ∑T'j ≤ ∑Tj (כי T'l≤Tl, T'b≤Tb, T'm≤Tl) אבל Tm אולי גדל → זה בסדר כי T'm ≤ Tl ו-T'l ≤ Tl, אז בסך הכל לא גרוע יותר. → S' לפחות טוב כמו S*. קיים אופטימום עם l לפני m. ∎
2) כש-dj = d לכל j, הבעיה 1|dj=d|∑Tj נפתרת פולינומיאלית
אם dj = d לכל j, אז לכל זוג l,m: dl = dm = d → dl ≤ dm ✓ אם בנוסף tl ≤ tm → לפי סעיף 1, l לפני m באופטימום. כלומר: סידור אופטימלי = מיון לפי tj עולה = SPT. מיון לוקח O(n log n) → פולינומיאלי. ∎
