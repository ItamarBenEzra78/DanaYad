תרגיל 3 - רגרסיה ללא חותך
────────────────────────────────────────────────────────────────

מודל: yᵢ = βxᵢ + εᵢ

א. המשוואה הנורמלית:
   ∂/∂β Σ(yᵢ - βxᵢ)² = 0
   
   ⟹ β̂ = Σxᵢyᵢ / Σxᵢ²

ב. הוכחה שהאומד מתפלג נורמלית:
   β̂ = Σxᵢyᵢ / Σxᵢ²
     = β + Σxᵢεᵢ / Σxᵢ²
   
   מאחר וεᵢ ~ N(0, σ²) בלתי מתואמים:
   β̂ ~ N(β, σ²/Σxᵢ²)

ג. שונות משותפת של ȳ ו-β̂₁:
   Cov(ȳ, β̂₁) ≠ 0 בדרך כלל
   
   סיבה: ȳ תלויה בכל εᵢ, ו-β̂ גם תלויה בכל εᵢ
   השיתוף בשגיאות יוצר מתאם שלילי בין שניהם

תרגיל 5 - השוואה בין שלושה אומדים
────────────────────────────────────────────────────────────────

מודל: yᵢ = βxᵢ + εᵢ (ללא חותך)

█ אומד (a): β̂ = (Σxᵢyᵢ - nxy̅) / (Σxᵢ² - nxy̅²)
  א. חסר הטיה? ✓ כן (OLS)
  ב. OLS? ✓ כן
  ג. שונות: σ²/Σ(xᵢ - x̄)²
  הערכה: ★★★ הטוב ביותר (BLUE)

█ אומד (b): β̂ = Σxᵢyᵢ / Σxᵢ²
  א. חסר הטיה? ✓ כן
  ב. OLS? ✓ כן (עבור רגרסיה ללא חותך)
  ג. שונות: σ²/Σxᵢ²
  הערכה: ★★☆ טוב כשנודעה אי-קיום חותך

█ אומד (c): β̂ = Σyᵢ / Σxᵢ
  א. חסר הטיה? ✗ לא (בדרך כלל יש הטיה)
  ב. OLS? ✗ לא
  ג. שונות: גבוהה מאוד
  הערכה: ★☆☆ לא טוב - בעל הטיה ושונות גדולה

מסקנה: אומד (a) הוא הטוב ביותר (Best Linear Unbiased Estimator - BLUE)
