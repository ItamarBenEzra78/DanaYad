תרגיל 1: קוביות והשקלה

נתונים

• 7 שקילות של כלי עם קוביות

• X (מספר קוביות): [4, 2, 2, 1, 4, 6, 3]

• Y (משקל בגרמים): [585, 412, 397, 303, 596, 814, 511]

א. המשמעות הפיסיקלית

  • β₀ (חותך בישרה): משקל הכלי עצמו (הטארה) - בגרמים
  • β₁ (שיפוע): משקל ממוצע של קובייה אחת - בגרמים

ב. אומדי OLS

β̂₁ = Σ(xᵢ - x̄)(yᵢ - ȳ) / Σ(xᵢ - x̄)² = 100.38 גרם
β̂₀ = ȳ - β̂₁·x̄ = 227.14 גרם

ג. אמידת שונות השגיאה

SSE = Σ(yᵢ - ŷᵢ)² = 671.97
σ̂² = SSE/(n-2) = 134.39 (גרם)²
σ̂ = 11.59 גרם

ד. בדיקת השערה: H₀: β₁ = 0 (α = 0.05)

סטטיסטיקה:
SE(β̂₁) = √(σ̂²/Sₓₓ) = 2.996
t-statistic = 100.38 / 2.996 = 33.53
p-value < 0.001
t_critical = 2.571 (df = 5)
✅ דחייה של H₀ - קיים קשר מובהק בין מספר הקוביות למשקל הכולל

תרגיל 2: אלכוהול וזמן תגובה

נתונים

• 10 נבדקים

• X (כוסות בירה): [0, 2, 4, 6, 1, 3, 5, 8, 4, 2]

• Y (הפרש זמן ms): [110, 210, 409, 262, 162, 57, 310, 210, 106, 9]

א. בדיקת קשר בין כמות בירה להפרש בזמן תגובה

β̂₁ = Sₓᵧ / Sₓₓ = 2292 / 52.5 = 43.66 ms/כוס
סטטיסטיקה:
SE(β̂₁) = √(σ̂²/Sₓₓ) = 6.67
t-statistic = 43.66 / 6.67 = 6.54
p-value < 0.001
✅ דחייה של H₀ - קיים קשר מובהק וחיובי בין כמות הבירה להפרש בזמן תגובה
כל כוס בירה נוספת מעלה את זמן התגובה בממוצע ב-43.66 מילישניות

ב. בדיקת השערה: H₀: β₀ = 0

חישובים:
β̂₀ = ȳ - β̂₁·x̄ = 32.19
SE(β̂₀) = √[σ̂²(1/n + x̄²/Sₓₓ)] = 32.31
t-statistic = 32.19 / 32.31 = 0.996
p-value = 0.342
❌ לא דחייה של H₀ - לא ניתן לדחות שהחותך שווה לאפס

משמעות בהקשר למחקר:

  • כאשר מספר כוסות הבירה = 0, הפער בזמן התגובה עשוי להיות 0
  • זה מעניין אך מפתיע מבחינה ביולוגית
  • התוצאה עלולה להיות תוצאת קו במדגם זה

תרגיל 3: רגרסיה ללא חותך

מודל: yᵢ = β·xᵢ + εᵢ
כאשר εᵢ ~ N(0, σ²) בלתי מתואמות

א. המשוואה הנורמלית

∂/∂β Σ(yᵢ - β·xᵢ)² = 0
⟹ β̂ = Σxᵢ·yᵢ / Σxᵢ²

ב. הוכחה שהאומד מתפלג נורמלית

β̂ = Σxᵢ·yᵢ / Σxᵢ² = β + Σxᵢ·εᵢ / Σxᵢ²
מאחר ו-εᵢ ~ N(0, σ²) בלתי מתואמות:
β̂ ~ N(β, σ²/Σxᵢ²)

ג. שונות משותפת של ȳ ו-β̂₁

לא בהכרח אפס!

  • ȳ תלויה בכל εᵢ (דרך ε̄)
  • β̂ גם תלויה בכל εᵢ
  • שיתוף הסיכון (common noise) זהה יוצר מתאם שלילי
