תרגיל 1: קוביות והשקלה
───────────────────────────────────────────────

נתונים:
• 7 שקילות של כלי עם קוביות
• X (מספר קוביות): [4, 2, 2, 1, 4, 6, 3]
• Y (משקל בגרמים): [585, 412, 397, 303, 596, 814, 511]

א. המשמעות הפיסיקלית

β₀ (חותך בישרה): משקל הכלי עצמו (הטארה) - בגרמים
β₁ (שיפוע): משקל ממוצע של קובייה אחת - בגרמים

ב. אומדי OLS

חישובים:
- n = 7
- x̄ = 3.0, ȳ = 528.29
- Σ(xᵢ - x̄)(yᵢ - ȳ) = 1506.57
- Σ(xᵢ - x̄)² = 15

תוצאות:
β̂₁ = 1506.57/15 = 100.38 גרם
β̂₀ = 528.29 - 100.38 × 3.0 = 227.14 גרם

ג. אמידת שונות השגיאה

SSE = Σ(yᵢ - ŷᵢ)² = 671.97
σ̂² = SSE/(n-2) = 671.97/5 = 134.39 (גרם)²
σ̂ = 11.59 גרם

ד. בדיקת השערה: H₀: β₁ = 0 (α = 0.05)

סטטיסטיקה:
- SE(β̂₁) = √(134.39/15) = 2.996
- t = 100.38 / 2.996 = 33.53
- p-value < 0.001
- t_critical = 2.571 (df = 5)

מסקנה:
✅ דחייה של H₀ - קיים קשר מובהק בין מספר הקוביות למשקל הכולל

תרגיל 3: רגרסיה ללא חותך
───────────────────────────────────────────────

מודל: yᵢ = β·xᵢ + εᵢ
כאשר ε ~ N(0, σ²) בלתי מתואמות

א. המשוואה הנורמלית

∂/∂β Σ(yᵢ - β·xᵢ)² = 0
⟹ β̂ = Σxᵢ·yᵢ / Σxᵢ²

ב. הוכחה שהאומד מתפלג נורמלית

β̂ = Σxᵢ·yᵢ / Σxᵢ² = β + Σxᵢ·εᵢ / Σxᵢ²
מאחר ו-εᵢ ~ N(0, σ²) בלתי מתואמות:
β̂ ~ N(β, σ²/Σxᵢ²)

התוחלת: E[β̂] = β
השונות: Var(β̂) = σ²/Σxᵢ²

ג. שונות משותפת של ȳ ו-β̂₁

לא בהכרח אפס!

ȳ = β·x̄ + ε̄
β̂ = β + Σ(xᵢ - x̄)·εᵢ / Σ(xᵢ - x̄)²

השכתוב הוא:
- ȳ תלויה בכל εᵢ (דרך ε̄)
- β̂ גם תלויה בכל εᵢ

מאחר ששיתוף הסיכון (common noise) זהה, יש מתאם בין שניהם.
מתאם שלילי: מאחר ש-β̂ מתותקן ב"עיוות" של השגיאות בכיוון x,
למשקל גדול, דרך שיווי משקל יוצר מתאם שלילי עם ȳ.

תרגיל 5: השוואת אומדים

מודל: yᵢ = β·xᵢ + εᵢ (ללא חותך)

השוואת האומדים:

אומד (a): β̂ = Σxᵢ·yᵢ - n·x̄·ȳ / Σxᵢ² - n·x̄²
- חסר הטיה? ✓ כן (OLS)
- אומד OLS? ✓ כן
- שונות: σ²/Sₓₓ (מינימלית)
- דירוג: ⭐⭐⭐ הטוב ביותר (BLUE)

אומד (b): β̂ = Σxᵢ·yᵢ / Σxᵢ²
- חסר הטיה? ✓ כן
- אומד OLS? ✓ כן (עבור מודל ללא חותך)

- דירוג: ⭐⭐ טוב כשודאים אין חותך

אומד (c): β̂ = Σyᵢ / Σxᵢ
- חסר הטיה? ✗ לא
- אומד OLS? ✗ לא
- שונות: גבוהה מאוד
- דירוג: ⭐ לא טוב

הוא Best Linear Unbiased Estimator (BLUE)
