# פתרון עבודת בית - רגרסיה לינארית

## תרגיל 1: קוביות והשקלה

### נתונים
- 7 שקילות של כלי עם קוביות
- X: [4, 2, 2, 1, 4, 6, 3]
- Y: [585, 412, 397, 303, 596, 814, 511]

### א. המשמעות הפיסיקלית

- **β₀ (חותך בישרה)**: משקל הכלי עצמו (הטארה) - בגרמים
- **β₁ (שיפוע)**: משקל ממוצע של קובייה אחת - בגרמים

### ב. אומדי OLS

**חישובים:**
- n = 7
- x̄ = 3.0, ȳ = 528.29
- Σ(xᵢ - x̄)(yᵢ - ȳ) = 1506.57
- Σ(xᵢ - x̄)² = 15

**תוצאות:**
$$\hat{\beta}_1 = \frac{1506.57}{15} = 100.38 \text{ גרם}$$

$$\hat{\beta}_0 = 528.29 - 100.38 \times 3.0 = 227.14 \text{ גרם}$$

### ג. אמידת שונות השגיאה

$$SSE = \sum(y_i - \hat{y}_i)^2 = 671.97$$

$$\hat{\sigma}^2 = \frac{SSE}{n-2} = \frac{671.97}{5} = 134.39 \text{ (גרם)}^2$$

$$\hat{\sigma} = 11.59 \text{ גרם}$$

### ד. בדיקת השערה: H₀: β₁ = 0 (α = 0.05)

**סטטיסטיקה:**
- SE(β̂₁) = √(134.39/15) = 2.996
- t = 100.38 / 2.996 = 33.53
- p-value < 0.001
- t_critical = 2.571 (df = 5)

**מסקנה:** 
✅ **דחייה של H₀** - קיים קשר מובהק ומובהק בין מספר הקוביות למשקל הכולל

## תרגיל 2: אלכוהול וזמן תגובה

### נתונים
- 10 נבדקים
- X (כוסות בירה): [0, 2, 4, 6, 1, 3, 5, 8, 4, 2]
- Y (הפרש זמן ms): [110, 210, 409, 262, 162, 57, 310, 210, 106, 9]

### א. בדיקת קשר בין כמות בירה להפרש בזמן תגובה

**חישובים:**
- Sₓₓ = 175 - 10(3.5)² = 52.5
- Sₓᵧ = 9117 - 10(3.5)(185) = 2292
- β̂₁ = 2292 / 52.5 = 43.66 ms/כוס
- σ̂² = 18680.8 / 8 = 2335.1

**סטטיסטיקה:**
- SE(β̂₁) = √(2335.1/52.5) = 6.67
- t = 43.66 / 6.67 = 6.54
- p-value < 0.001
- t_critical = 2.306 (df = 8)

**מסקנה:** 
✅ **דחייה של H₀** - קיים קשר מובהק וחיובי בין כמות הבירה להפרש בזמן תגובה

כל כוס בירה נוספת מעלה את זמן התגובה בממוצע ב-43.66 מילישניות.

### ב. בדיקת השערה: H₀: β₀ = 0

**חישובים:**
- β̂₀ = 185 - 43.66(3.5) = 32.19
- SE(β₀) = √(2335.1 × (1/10 + 3.5²/52.5)) = 32.31
- t = 32.19 / 32.31 = 0.996
- p-value = 0.342

**מסקנה:** 
❌ **לא דחייה של H₀** - לא ניתן לדחות שהחותך שווה לאפס

**משמעות בהקשר למחקר:**
- כאשר מספר כוסות הבירה = 0 (מצב בסיסי), הפער בזמן התגובה עשוי להיות 0
- זה מעניין אך מפתיע מבחינה ביולוגית - בדרך כלל מצפים למינוס בסיסי בפיגור גם ללא אלכוהול
- התוצאה עלולה להיות תוצאת קו במדגם זה

## תרגיל 3: רגרסיה ללא חותך

### מודל
$$y_i = \beta x_i + \varepsilon_i$$

כאשר ε ~ N(0, σ²) בלתי מתואמות

### א. המשוואה הנורמלית

$$\frac{\partial}{\partial \beta} \sum_{i=1}^n (y_i - \beta x_i)^2 = 0$$

$$-2\sum_{i=1}^n x_i(y_i - \beta x_i) = 0$$

$$\hat{\beta} = \frac{\sum_{i=1}^n x_i y_i}{\sum_{i=1}^n x_i^2}$$

### ב. הוכחה שהאומד מתפלג נורמלית

**הוכחה:**
$$\hat{\beta} = \frac{\sum x_i y_i}{\sum x_i^2} = \frac{\sum x_i(\beta x_i + \varepsilon_i)}{\sum x_i^2}$$

$$= \beta + \frac{\sum x_i \varepsilon_i}{\sum x_i^2}$$

מאחר ו-εᵢ ~ N(0, σ²) בלתי מתואמות, סכומן הלינארי הוא גם נורמלי.

$$\hat{\beta} \sim N\left(\beta, \frac{\sigma^2}{\sum x_i^2}\right)$$

**התוחלת:**
$$E[\hat{\beta}] = \beta$$

**השונות:**
$$\text{Var}(\hat{\beta}) = \frac{\sigma^2}{\sum x_i^2}$$

### ג. שונות משותפת של ȳ ו-β̂₁

**לא בהכרח אפס!**

$$\bar{y} = \beta \bar{x} + \bar{\varepsilon}$$

$$\hat{\beta} = \beta + \frac{\sum(x_i - \bar{x})\varepsilon_i}{\sum(x_i - \bar{x})^2}$$

**השכתוב הוא:**
- ȳ תלויה בכל εᵢ (דרך ε̄)
- β̂ גם תלויה בכל εᵢ

מאחר ששיתוף הסיכון (common noise) זהה, יש מתאם בין שניהם.

**מתאם שלילי:** מאחר ש-β̂ מתותקן ב"עיוות" של השגיאות בכיוון x, למשקל גדול, דורך שיווי משקל יוצר מתאם שלילי עם ȳ.

## תרגיל 4: השוואת מכונות ייצור

### נתונים
- 6 מחזורי ייצור
- X (חומר גלם ק"ג): [4, 6, 8, 7, 2, 5]
- מכונה א': [67, 101, 150, 131, 24, 95]
- מכונה ב': [74, 113, 159, 136, 23, 94]

### א. משמעות הפרמטרים

**מכונה א':** y[(a)]ᵢ = α₀ + α₁xᵢ + δᵢ
- **α₀**: מספר בסיסי של סביבונים כשלונים/פגומים (קיבול של A-טארה)
- **α₁**: כמה סביבונים נוספים מ-1 ק"ג חומר גלם
- **δᵢ**: סטייה בתצפית i מהמודל

**מכונה ב':** כנ"ל עם β₀, β₁, εᵢ

### ב. בדיקת השערה: H₀: σ²ₐ = σ²ᵦ vs H₁: σ²ₐ > σ²ᵦ

**אמידות:**
- מכונה א': σ̂²ₐ = 290.53
- מכונה ב': σ̂²ᵦ = 280.03

**F-statistic:**
$$F = \frac{\sigma^2_a}{\sigma^2_b} = \frac{290.53}{280.03} = 1.0376$$

**F_critical** (α = 0.05, df₁ = 4, df₂ = 4) = 6.388

**מסקנה:** 
❌ **לא דחייה** - לא קיים הבדל מובהק בשונויות

### ג. קביעת טווחי שימוש

**משוואות רגרסיה (כפי שניתנו בתרגיל):**
- מכונה א': ŷ = -16.1143 + 20.77x
- מכונה ב': ŷ = -18.87 + 22.25x

**נקודת הצומת** (y_A = y_B):
$$-16.1143 + 20.77x = -18.87 + 22.25x$$
$$2.7557 = 1.48x$$
$$x = 1.862 \text{ ק"ג}$$

**סיפור:**
- **x < 1.86 ק"ג**: תשמשו **מכונה א'** (מספר סביבונים גבוה יותר)
- **x > 1.86 ק"ג**: תשמשו **מכונה ב'** (יעילות טובה יותר)

## תרגיל 5: השוואה בין שלושה אומדים

### מודל
$$y_i = \beta x_i + \varepsilon_i$$
ללא חותך

### השוואת האומדים

#### אומד (a): $\hat{\beta} = \frac{\sum x_i y_i - n\bar{x}\bar{y}}{\sum x_i^2 - n\bar{x}^2}$

| קריטריון | תשובה |
|---------|--------|
| **חסר הטיה?** | ✓ כן (OLS) |
| **אומד OLS?** | ✓ כן |
| **שונות** | σ²/Σ(xᵢ - x̄)² (מינימלית) |
| **דירוג** | ⭐⭐⭐ הטוב ביותר (BLUE) |

#### אומד (b): $\hat{\beta} = \frac{\sum x_i y_i}{\sum x_i^2}$

| קריטריון | תשובה |
|---------|--------|
| **חסר הטיה?** | ✓ כן |
| **אומד OLS?** | ✓ כן (עבור מודל ללא חותך) |
| **שונות** | σ²/Σxᵢ² |
| **דירוג** | ⭐⭐ טוב כשודאים אין חותך |

#### אומד (c): $\hat{\beta} = \frac{\sum y_i}{\sum x_i}$

| קריטריון | תשובה |
|---------|--------|
| **חסר הטיה?** | ✗ לא |
| **אומד OLS?** | ✗ לא |
| **שונות** | גבוהה מאוד |
| **דירוג** | ⭐ לא טוב |

### מסקנה

**אומד (a) הוא הטוב ביותר!** 

הוא הוא Best Linear Unbiased Estimator (BLUE) בעל:
- ✓ אי-הטיה
- ✓ תוכנית OLS
- ✓ שונות מינימלית
